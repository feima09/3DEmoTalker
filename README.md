# 3DEmoTalker 虚幻引擎数字人项目
## 项目概述
3DEmoTalker 是一个基于虚幻引擎5.3开发的智能数字人项目，集成了语音识别、语音合成、AI对话和3D面部动画驱动技术。项目包含完整的UE5工程文件、后端服务接口和算法模块，支持本地部署和云端API接入。

<video src="doc/demo.mp4" controls autoplay loop width="100%"></video>

## 主要特性

🗣️ 多语言语音识别（FunASR）

🔊 高质量语音合成（MeloTTS）

💬 大语言模型对话（支持阿里API/本地部署）

👄 实时面部动画驱动（NVIDIA Audio2Face）

🎮 UE5实时渲染数字人

🌐 WebUI配置面板
